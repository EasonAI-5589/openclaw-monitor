### 🚀 OpenClaw 本体

**最新版本: v2026.2.15** (发布于 2026-02-16)
- Beta: `v2026.2.15-beta.1`

#### 重要新功能
- **Discord Components v2** — 支持按钮、选择菜单、模态框、附件文件块，实现原生 Discord 交互体验
- **嵌套子代理 (Sub-sub-agents)** — 可配置深度 `maxSpawnDepth`，子代理可以再生成子代理
- **LLM Hook 扩展** — 暴露 `llm_input` / `llm_output` 钩子，插件可观察 prompt 上下文和模型输出
- **Cron Webhook** — 支持完成运行的 webhook 投递和独立认证 token

#### 安全修复 (重要!)
- 🔒 SHA-1 → SHA-256 沙箱配置哈希
- 🔒 Telegram bot token 从错误日志中脱敏
- 🔒 阻止危险的 Docker 沙箱配置（bind mount、host 网络等）
- 🔒 Skill download 安装器限制 targetDir 防止任意文件写入
- 🔒 Web Fetch 响应体大小上限防止内存耗尽
- 🔒 Control UI 防 XSS 注入
- 🔒 工作区路径 Unicode 注入防护

#### 其他修复
- Telegram 流式预览不再产生重复消息
- Memory FTS 支持 CJK 等 Unicode 查询
- 群聊每轮都注入群组上下文（不再丢失群组感知）
- `NO_REPLY` 不再在 TUI/Web 显示占位符
